# Invoice Management System

This is a task to  build the API for selected pages of an invoice management system.


## Tech Stack Used
- Go-Gin (Golang API Framework)
- Postgres (Data store)
- Golang-Migrate (for Database migration)
- Sqlc (For ORM)
- Make (for Project Build)
